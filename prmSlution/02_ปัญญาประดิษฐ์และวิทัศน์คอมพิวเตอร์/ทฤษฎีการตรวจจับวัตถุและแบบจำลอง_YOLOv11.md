# ทฤษฎีการตรวจจับวัตถุและแบบจำลองปัญญาประดิษฐ์ (Object Detection via YOLOv11)

แกนหลักสำคัญในการสกัดข้อมูลละอองเม็ดน้ำยาสารเคมีจากแผนกระจกสไลด์ภายใต้กล้องจุลทรรศน์ คือความสามารถในการตรวจจับเชิงปริมาณอย่างแม่นยำ โครงการนี้ได้เลือกสรรสถาปัตยกรรมปัญญาประดิษฐ์ในตระกูล **YOLO (You Only Look Once)** เวอร์ชันล่าสุด (YOLOv11) ในการสกัดพิกัดขอบเขตและรูปแบบคอนทัวร์ (Contour) ด้วยกระบวนทัศน์ทางทฤษฎีดังนี้:

## แผนภาพแสดงสถาปัตยกรรมการตรวจจับละอองเม็ดน้ำยาสารเคมี

```mermaid

    A[รับภาพจากกล้องจุลทรรศน์] --> B[Preprocessing<br/>ปรับขนาดภาพเป็น 640x640 Pixel]
    B --> C[YOLOv11 ONNX Runtime Inference<br/>โครงข่ายประสาทเทียมแบบ Deep Learning]
```

## 1. การอนุมานโมเดลผ่านสถาปัตยกรรม ONNX Runtime
เพื่อให้ซอฟต์แวร์ทำงานได้รวดเร็วระดับเวลาจริงบนเครื่องลูกข่าย (Edge Computing Process) โมเดลต้นฉบับจำลอง (Weights) ได้ถูกแปลงออกเป็นรูปแบบตระกูลสากลคือ **ONNX (Open Neural Network Exchange)** รูปแบบนี้ทำให้โปรแกรม C# ที่รองรับผ่าน `Microsoft.ML.OnnxRuntime` ดึงความสามารถของกราฟฟิกการ์ดหรือ CPU ภายในตัวเครื่องเดสก์ท็อปพีซีได้อย่างเต็มสมรรถนะ 

## 2. ขั้นตอนการสกัดและการแปลงค่าเทนเซอร์ (Tensor Manipulation)
1. **กระบวนการปรับภาพก่อนเข้าแบบจำลอง (Preprocessing):** เฟรมวิดีโอสี (RGB) จะถูกลดทอนและจัดรูปให้เป็นสี่เหลี่ยมจัตุรัสขนาด 640x640 พิกเซล (มิติ Tensor 1x3x640x640) และปรับสเกลความสว่าง (Normalization) เป็นระหว่างช่วงพิกัดสเกลาร์ 0.0 - 1.0 
2. **เครือข่ายประสาทลึก (Deep Neural Network Inference):** โยนโครงข่ายลูกเต๋า Tensor เข้าสู่ Convolution Layer เพื่อสกัดหาลักษณะเด่น (Features Maps) ออกมาให้ได้เป็นชุดข้อมูล Bounding Box (พิกัดกรอบ XY) พร้อมความมั่นใจความเป็นละอองเป้าหมาย (Confidence Score) 
3. **การคัดกรองสัญญาณรบกวน (Thresholding & Area Filtering):** ละอองเม็ดน้ำยาสารเคมีที่ให้ค่าความมั่นใจที่ต่ำกว่าค่าวิกฤติด้านผู้ใช้งาน (เช่น ค่าเฉลี่ยต่ำกว่า 0.25) จะถูกปัดตกในทันที ระบบป้องกันความสนใจที่ผิดพลาดโดยการตรวจสอบความยาวด้านกว้างxสูง ที่มีขนาดใหญ่ผิดปกติ (เช่นเกินกว่าอัตรา 80% ของหน้าจอ) เพื่อป้องกันโมเดลจับแสงสะท้อนขยะหรือขอบรอยแตกของกระจก.

## 3. อัลกอริทึมระงับวัตถุซ้อนทับ (Non-Maximum Suppression - NMS)
การประมวลผลโมเดล AI มักสร้างกรอบผลลัพธ์รอบตัวละอองเม็ดน้ำยาสารเคมี 1 จุดเกิดเป็นจำนวน 2-5 กรอบซ้อนกัน. โครงการอิงใช้เทคนิคเชิงคณิตศาสตร์ที่ชื่อว่า "Non-Maximum Suppression" เพื่อทำหน้าที่แก้ปัญหาความซ้ำซ้อน:
- โดยนำเอาพิกัดกล่องทั้งหมดมาเปรียบเทียบหาพื้นที่การซ้อนทับกัน ทางคณิตศาสตร์เรียกว่าสูตร Intersection over Union (IoU)
- หากกล่อง A ทับซ้อนกับกล่อง B ในอาณาบริเวณที่เกินร้อยละ 45 (IoU > 0.45) นำคะแนนความเชื่อมั่น (Confidence) มาเป็นเกณฑ์ตัดสินใจ: ตัวไหนคะแนนต่ำกว่า จะโดนลบทิ้ง (Suppression)
ด้วยสมการดังกล่าว เราจะรีดภาพจาก 5 กรอบรกๆ ให้เป๊ะเหลือเพียง 1 กรอบกล่อง ต่อ 1 ละอองเม็ดน้ำยาสารเคมี ได้อย่างแม่นยำที่สุด

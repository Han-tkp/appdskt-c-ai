# ระบบการติดตามพิกัดวัตถุในระบบพิกัดคาร์ทีเซียน XY (XY-Coordinate Object Tracking)

ปัญหาท้าทายสำคัญในการนำเข้าผลลัพธ์การตรวจจับภาพละอองแบบเฟรมต่อเฟรมในรูปแบบของวิดีโอสตรีมมิ่ง คือปัญหา **"การจำสลับตัว"** เครื่องจะคิดว่าละอองเม็ดน้ำยาสารเคมี 1 เม็ดในเฟรมที่ 1 เป็นคนละตัวกับเม็ดในเฟรมที่ 2 ทำให้เกิดการนับเบิ้ลสะสมตัวเลขขึ้นเร็วเกินจริงเมื่อผู้ใช้งานขยับหน้ากล้องจุลทรรศน์ ด้วยปัญหาดังกล่าว โครงการได้พัฒนาระบบฝังตัว "ตัวติดตามออปเจกต์" (Object Tracker) เชิงคณิตศาสตร์และเรขาคณิตด้วยสถาปัตยกรรมแบบ **SORT-Lite (Simple Online and Realtime Tracking)** โดยอิงกระบวนการเชิงลึกดังนี้:

## แผนภาพแสดงกระบวนการติดตามและเชื่อมโยงรหัสระบุตัวตน (Tracking ID Matching)

```mermaid

        B(ละอองเม็ดน้ำยาสารเคมี ID #2<br/>พิกัด X2, Y2)
    end
```

## 1. จุดศูนย์กลางมวลในระบบคาร์ทีเซียนแกน XY (Centroid Euclidean Calculation)
เมื่อปัญญาประดิษฐ์คืนค่ารูปทรงเรขาคณิต (Contour) ของละอองเม็ดน้ำยาสารเคมี โปรแกรมจะใช้สูตรหาจุดกึ่งกลางมวลเสมือน (Moments) ของรูปร่างนั้นๆ ออกมาเป็นพิกัด $(X, Y)$ ประจำตัว
- รักษาสถานะทางความจำ: สร้างแผนที่ตัวแปรให้วัตถุในเฟรมก่อนหน้าเก็บค่าแกนพิกัด $X_1, Y_1$ และเส้นผ่านศูนย์กลางขนาด $D_1$ เอาไว้ตลอดอายุวัฏจักร

## 2. สมการจับคู่ข้ามเฟรมโดยอิงระยะกระจัดยูคลิด (Euclidean Distance Matching)
เมื่อระบบรับเฟรมหน้ากล้องสแกนล่าสุดที่มิลลิวินาทีถัดมา มันจะคำนวณตำแหน่งวัตถุต่างๆ ด้วยค่าพิกัด $X_2, Y_2$ วิธีการเชื่อมโยงสายสัมพันธ์เพื่อหาตัวสืบทอดร่าง (Identity) กระทำโดยสมการการหาระยะจัดระหว่างจุด 2 จุด (Euclidean Distance):

$Distance = \sqrt{(X_2 - X_1)^2 + (Y_2 - Y_1)^2}$

หากค่า $Distance$ มีความห่างน้อยกว่าค่าขีดวิกฤติที่กำหนดไว้ (เช่น < 80 Pixel Error Radius) ระบบเริ่มพิจารณาสมมติฐานว่า ออปเจกต์ 1 ในอดีต และ 2 ในสตรีมปัจจุบัน น่าจะเป็นละอองเม็ดน้ำยาสารเคมีเม็ดเดียวกันที่เพียงแค่กระเพื่อมไหวจากการเคลื่อนสไลด์

## 3. มิติโครงสร้าง (Diameter Validation Rate)
เพื่อรัดกุมการจดจำผิดพลาดในบริเวณที่มีละอองเม็ดน้ำยาสารเคมีกระจุกตัวหนาแน่น ระบบได้เพิ่มเงื่อนไขการตรวจสอบอีกหนึ่งชั้นคือพารามิเตอร์ของ **ความกลมกลืนด้านขนาดเส้นผ่านศูนย์กลาง (Size Ratio Validation)**: โดยการนำเอาพิกเซลขนาดของละอองเก่ากับละอองใหม่มากางเปรียบเทียบหาอัตราส่วน หากสัดส่วนเกินกำหนด (เช่นเกินกว่าอัตรา 1.5 เท่าตัวของกันและกัน) แปลว่านี่น่าจะเป็นละอองใหม่ที่เกิดมารวมตัวทับซ้อนกันในตำแหน่งเดิม ไม่ใช่วัตถุเป้าหมายตัวเก่า 
เมื่อผ่านกระบวนการคัดกรอง 2 ด่านหลักนี้ ละอองจะได้รับหมายเลขบัตรประชาชนเสมือน (Tracking ID Code) รอดข้ามจากเฟรมหนึ่งไปสู่อีกเฟรมหนึ่ง ซึ่งช่วยลดระดับความทับซ้อนซ้ำในการนับมวลน้ำของตัวแปรสะสมคะแนนได้อย่างหมดจด.

## 4. ความทนทานการเคลื่อนหลุดขวาจอชั่วคราว (Occlusion / Missing Frame Tolerance)
เพื่อเสริมเสถียรภาพ ระบบได้จัดกำหนดกรอบระยะเวลาผ่อนผันการค้นหาตัว (Missing Frames Tolerance Limit = 5 Frames) เพื่อรักษาพยุงสถานะเลข Tracking ID เอาไว้ ในเหตุการณ์ที่ AI กระพริบมองไม่เห็นเศษเสี้ยววินาที หรือละอองตกขอบจอแล้วถูกเลื่อนกลับเข้ามาอย่างรวดเร็ว ข้อมูลจะไม่ถูกสั่งทำลายล้างสถานะทันทีในทันใด ทำให้รหัส Tracking ID ไม่หลุดร่วงบ่อยครั้งจนน่ารำคาญ.
